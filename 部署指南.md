# 🚀 快速部署指南

## 一键部署到Netlify

### 方式一：使用Netlify CLI（推荐）

1. **安装Netlify CLI**
   ```bash
   npm install -g netlify-cli
   ```

2. **登录Netlify**
   ```bash
   netlify login
   ```

3. **创建新站点**
   ```bash
   # 在项目根目录执行
   netlify init
   
   # 按照提示：
   # - 选择 "Create & configure a new site"
   # - 输入站点名称
   # - 设置发布目录: public
   # - 设置函数目录: netlify/functions
   ```

4. **配置KV数据库**
   ```bash
   # 启用KV存储
   netlify addons:create kv
   
   # 获取环境变量
   netlify addons:kv:list
   ```

5. **配置环境变量**
   ```bash
   # 将获取到的URL和Token配置为环境变量
   netlify env:set KV_REST_API_URL "your_kv_url"
   netlify env:set KV_REST_API_TOKEN "your_kv_token"
   ```

6. **部署**
   ```bash
   netlify deploy --prod
   ```

### 方式二：使用Netlify网页版

1. **访问Netlify**
   - 打开 https://app.netlify.com/
   - 登录你的账号

2. **导入Git仓库**
   - 点击 "Add new site" → "Import an existing project"
   - 连接你的Git提供商（GitHub/GitLab/Bitbucket）
   - 选择本项目的仓库

3. **配置构建设置**
   ```
   Build command: (留空)
   Publish directory: public
   Functions directory: netlify/functions
   ```

4. **部署**
   - 点击 "Deploy site"
   - 等待部署完成

5. **配置KV数据库（部署后）**
   - 进入 "Site settings" → "Build & deploy" → "Environment"
   - 点击 "Edit variables"
   - 添加KV环境变量（需要先启用KV插件）

### 方式三：拖拽部署（最简单）

1. **打包项目**
   ```bash
   # 确保所有文件都在项目文件夹中
   # 准备好 public 和 netlify 文件夹
   ```

2. **拖拽上传**
   - 登录Netlify
   - 在 "Sites" 页面，拖拽整个项目文件夹到页面
   - 等待自动部署

## 🔧 配置KV数据库

### 通过Netlify网页配置

1. 进入 "Site settings"
2. 点击 "Enable Netlify KV"
3. 等待KV数据库创建完成
4. 点击 "Create variable"
5. 添加两个环境变量：
   - `KV_REST_API_URL`: 复制KV提供的URL
   - `KV_REST_API_TOKEN`: 复制KV提供的Token

### 通过CLI配置

```bash
# 启用KV
netlify addons:create kv

# 查看KV信息
netlify addons:kv:list

# 设置环境变量（替换为实际值）
netlify env:set KV_REST_API_URL "https://your-account.kb.us-east-1.aws.kv.netlify.app"
netlify env:set KV_REST_API_TOKEN "your_token_here"
```

## ⚡ 性能优化

### 图片优化
- 使用CDN加速图片加载
- 压缩图片文件大小
- 使用WebP格式（现代浏览器）

### 缓存配置
在 `netlify.toml` 中添加：
```toml
[[headers]]
  for = "/*"
  [headers.values]
    Cache-Control = "public, max-age=3600"

[[headers]]
  for = "/images/*"
  [headers.values]
    Cache-Control = "public, max-age=31536000"
```

### 启用压缩
Netlify自动启用Gzip和Brotli压缩，无需额外配置。

## 🐛 常见问题

### Q: 部署后无法连接数据库？
A: 检查环境变量是否正确配置，并确保已重新部署

### Q: 图片上传失败？
A: 检查图片大小和格式，确保不超过10MB

### Q: AI审核总是失败？
A: 这是模拟系统，可以修改代码中的随机概率

### Q: 如何修改管理员账号？
A: 在 `public/js/app.js` 中修改 `admin` 用户名检查
